Real-Time Fraud Detection System - Package Notes

This zip contains the portfolio project code, documentation, tests, dashboard,
API, model artifacts, and generated monitoring artifacts.

Included:
- app.py
- src/
- tests/
- docs/PROJECT_BREAKDOWN.html
- README.md
- requirements.txt
- Dockerfile and docker-compose.yml
- models/ artifacts generated during local training
- data/processed/ if present

Not included:
- venv/ virtual environment
- __pycache__/ folders
- data/raw/creditcard.csv
- mlruns/ local MLflow tracking history

Why the raw dataset is excluded:
The creditcard.csv dataset is a Kaggle dataset. Keep your local copy in:
data/raw/creditcard.csv

Beginner run order:
1. Open Command Prompt.
2. cd /d "C:\Users\admin\Desktop\Real-Time Fraud Detection System"
3. venv\Scripts\activate.bat
4. python -m src.processing.feature_engineering
5. python -m src.modeling.train_batch
6. python -m src.modeling.threshold_tuner
7. Start FastAPI: uvicorn src.api.main:app --host 127.0.0.1 --port 8001 --reload
8. Start Streamlit in another Command Prompt: streamlit run app.py --server.port 8502
9. Open http://127.0.0.1:8502 and click Generate Demo Alerts if the Live Feed is empty.
